## Steps
### Step 1. Create a django project
1. Create a virtual environment by running the following command:
  ```
  python -m venv env
  ```
2. Activate the virtual environment by running this command:
  - Windows:
    ```
    env\Scripts\activate.bat
    ```
  - Mac:
    ```
    source env/bin/activate
    ```
